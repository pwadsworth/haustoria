# Haustoria

**Haustoria** is a 2D puzzle-platformer where players navigate through challenging environments using physics-based interactions, combat, and resource recovery mechanics. 

Built using Python and the Arcade library, this game features a custom sub-stepped physics engine, destructible terrain, state-machine driven enemy AI, and interactive throwable objects.

## Running the Game
To run the prototype vertical slice, ensure you have Python installed with the Arcade library.
```bash
cd haustoria
python main.py
```
